# SCT_SD_2 - Number Guessing Game

## Description
This is a GUI-based Number Guessing Game developed using Python. The application generates a random number between 1 and 100, and the user must guess the correct number. The program provides feedback indicating whether the guess is too high, too low, or correct.

## Features
- Graphical User Interface (GUI) using Tkinter
- Random number generation using the Random module
- User input through an entry field
- Feedback for high and low guesses
- Input validation using try-except
- Attempt counter
- Best score tracking
- Play Again button to restart the game

## Technologies Used
- Python
- Tkinter
- Random Module

## How to Run
1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the following command:

```bash
python main.py
```
